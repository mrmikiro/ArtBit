# ArtBit

Personal art collection management app with ultra-minimalist design.

## Features

- **Minimalist Gallery Grid**: Borderless grid displaying your art collection
- **Smart Search**: Search bar with advanced filter chips (Author, Technique, Modality)
- **Artwork Registration**: Title, Author, Modality, Technique, Movement, Value
- **Camera/Gallery Integration**: Upload artwork photos
- **Smooth Transitions**: Micro-interactions and Hero animations
- **Local Storage**: SQLite database for offline persistence

## Design

- Color palette: Deep greys, off-whites, matte black accents
- Typography: Inter (sans-serif)
- Generous whitespace for visual hierarchy
- Ultra-thin borders (0.5px) instead of heavy shadows

## Tech Stack

- **Framework**: Flutter
- **State Management**: Provider
- **Database**: SQLite (sqflite)
- **Architecture**: Clean separation (models, providers, screens, widgets)

## Getting Started

```bash
flutter pub get
flutter run
```
